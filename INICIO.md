# Guía de Inicio Rápido

## Pasos para ejecutar el proyecto

### 1. Instalar Node.js y npm
Asegúrate de tener Node.js (v18 o superior) instalado:
```bash
node --version
npm --version
```

### 2. Instalar dependencias
En el directorio raíz del proyecto, ejecuta:
```bash
npm install
```

### 3. Iniciar el servidor de desarrollo
```bash
npm start
```

O también puedes usar:
```bash
ng serve
```

### 4. Abrir en el navegador
Una vez que el servidor esté corriendo, abre tu navegador en:
```
http://localhost:4200
```

## Características del Dashboard

### Vista en Tiempo Real
- Los sensores se actualizan automáticamente cada 3 segundos
- Indicadores visuales de estado (Normal, Alerta, Crítico)
- Barras de progreso animadas
